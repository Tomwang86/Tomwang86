## 📖 项目简介

本项目为**2024年11月27日冬季消防安全主题班会**准备的图文并茂演示文稿，旨在帮助各班级开展针对性强、实效性高的安全教育活动。

## 🎯 班会主题

**关心关爱 · 平安校园 · 健康生活**

## 📋 内容涵盖

### 🔥 消防安全教育
- ❄️ 冬季火灾高发原因分析
- 🔍 校园火灾隐患排查要点
- 🧯 消防器材正确使用方法
- 🚪 火灾逃生技巧与自救
- ⚠️ 典型火灾案例警示教育

### 💚 身心健康引导
- 😴 规律作息的重要性
- 🥗 均衡饮食营养建议
- ⏰ 学校作息时间要求（24:00熄灯）
- 💪 充足睡眠健康保障

## 🔥 重点强调

### ⚠️ 消防安全"六个严禁"
1. 严禁使用大功率电器
2. 严禁私拉乱接电线
3. 严禁在宿舍吸烟
4. 严禁使用明火
5. 严禁堆放易燃物品
6. 严禁离开时不断电

### ⏰ 作息时间"铁律"
- 每晚 **24:00准时熄灯**
- 保证 **8小时充足睡眠**
- 熄灯后不使用电子设备
- 离开宿舍务必断电
